Open the Gazzetta-style energy dashboard:

[https://varco-demo.andreabaccega.com/](https://varco-demo.andreabaccega.com/)

The demo is a browser-only consumer backed by a synthetic Home Assistant showcase instance. It connects through Varco with a pre-approved read-only grant for only the energy entities used by the dashboard.

The demo is useful for understanding the consumer side: it behaves like a standalone app, but it does not receive a Home Assistant token.
